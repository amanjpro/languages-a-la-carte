trait ConstructorsCheckerComponent extends CheckerComponent[(Tree, ConstructorCheckerEnv)]

This phase checks the correctness of constructors, by:

  • Makes sure that all final fields are initialized after all constructors
  • Makes sure that constructs do not have a circular dependency on each other
  • Makes sure that no field is used before the explicit constructor invocation
  • Makes sure that final fields are not initialized twice.
