  52. trait InitializerComponent extends TransformationComponent[Tree, Tree]

    This phase lowers the AST by eliminating all static initializers and introducing a clinit.

    This phase lowers the AST by eliminating all static initializers and introducing a clinit. Then it move all the field initializers that are not static and literal to their respective initializer methods (namely either constructors or clinit). Initializers will not be added to the constructors that explicitly invoke another local constructor (not super constructor), this way we avoid accidentally initializing a field twice.
