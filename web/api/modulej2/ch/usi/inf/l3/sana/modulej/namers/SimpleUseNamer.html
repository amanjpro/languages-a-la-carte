  17. def fromQualifiedString(name: String, use: SimpleUseTree): UseTree

    Given a fully-qualified name and a SimpleUseTree, it returns a fully qualified tree of which resembles the fully qualified name but with all the attributes of the instance of the SimpleUseTree.

    Given a fully-qualified name and a SimpleUseTree, it returns a fully qualified tree of which resembles the fully qualified name but with all the attributes of the instance of the SimpleUseTree.
